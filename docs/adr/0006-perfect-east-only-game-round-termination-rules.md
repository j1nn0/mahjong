# ADR 0006: Perfect East-Only Game Round Termination Rules

## Context

以下の3つのエッジケースが未修正の状態にあった。

1. **途中流局時の親の移動**: 途中流局（九種九牌・四風連打・四家立直・四槓散了・三家和）が発生した場合に `dealerContinues = false` となっており、一般的なリーチ麻雀の標準ルールに反して親が次席へ移動していた。
2. **南4局での親連荘によるゲーム継続**: 南4局でトップが30000点未満のまま親が連荘した場合に、`roundNumber >= 4` の条件だけでゲームが強制終了していた。親連荘が本来続行できるケースが欠落していた。
3. **途中流局時のサドンデス判定**: 南場でトップが30000点以上の状態で途中流局が発生した場合、点数移動がないにも関わらずサドンデス終了していた。

## Decision

### 1. 途中流局時は常に親継続（連荘）
`finishAbortiveDraw` において `dealerContinues = true`、`isAbortiveDraw = true` を指定する。これにより途中流局後は必ず同じプレイヤーが親を続け、本場が1つ増える。

### 2. 南4局の強制終了条件に `!dealerContinues` を追加
```
} else if (state.roundNumber >= 4 && !dealerContinues) {
  matchEnded = true; // 南4局終了で強制打ち切り（親が流れた場合のみ）
}
```
これにより、南4局で親が連荘しトップが30000点未満であれば「南4局1本場」以降も続行できる。

### 3. 途中流局時はサドンデス判定をスキップ
`finishRound` に `isAbortiveDraw?: boolean` 引数を追加し、`true` の場合はトビ判定のみを行い、30000点によるサドンデス終了や局数による強制終了を行わない。

## Consequences

- 東風戦のゲーム終了判定がリーチ麻雀の標準ルールに完全に準拠した。
- 途中流局が連続して発生するケース（例: 南場で毎局九種九牌）でもゲームが正常に続行できる。
- 既存テスト（途中流局後のディーラー変数アサーション）を親継続を反映した形に修正した。
- 新規テストを4件追加し、南4局親連荘継続・途中流局時の挙動を網羅した。
