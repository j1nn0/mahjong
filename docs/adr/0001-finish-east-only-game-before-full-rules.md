# Finish East-Only Game Before Full Rules

The immediate completion target is an east-only game that can be played from East 1 through the final hand with stable round progression, scoring movement, riichi sticks, honba, dealer continuation, saved-game restore choice, and final ranking. We deliberately do not treat full rule coverage, all yaku edge cases, uma/oka, furiten, or special abortive draws as part of this target because those rule details are easier to add after the game can reliably complete a whole match.
